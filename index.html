Lesson 1: The Basics of the Web & HTML

How the Web Works

The world wide web is a collection of Html documents accessed via the internet. When a person accesses a web browser such as internet explorer or firefox to access a webpage i.e yahoo.com, their computer sends a HTTP request to a server. The server then delivers the web page to the web browser via the Internet. This HTML document is then displayed on the user's computer.

Browser

A browser is known as a computer program used to find a view information on the Internet

HTTP

"Hypertext Transfer Protocol" better known as HTTP, is the main protocol of the web. The browser uses Http to communicate with the servers.

Server

A Server is like a little apartment housing all the webpages. The server interprets the Html tags and sends them back as a webpage to our browser.

HTML

"Hyper Text Markup Language" better known as HTML, is the basis for almost every webpage

Tags & Elements

Tags are always contained within an angled bracket. An element is everything within a set of opening and closing tags. Html elements are either inline or block. Block just means there is an invisible box around the content.The invisible block can be a certain height and width

Attributes

Attributes are anchors used for referencing hyper links

Lesson 2: Creating a Structured Document w/ HTML

Document Object Model (DOM)

Document Object Model is referred to a tree-like structure of a page. Viewing this structure in a browser, it will show as a series of nested boxes.

Boxes

Everything on a webpage is in a box. There are also boxes within boxes. The more indented an element is, the deeper the nest box.

Text Editor

Writing a code can be difficult. Text editing programs like Sublime Text to assist in code writing. Sublime Text will automatically generate a closing HTML tag when you input an opening tag.

Lesson 3: CSS

CSS

Cascading Style Sheets - style sheet language used for describing the look and formatting of a document written in a markup language.

Repetition

Repetition is one of the main things you have to look out for. People have a tendency to make mistakes and not be consistant.
